winebot

commands:
'ping
- checks the bot's server latency
'kick [USER] <REASON>
- kicks a user
'ban [USER] <REASON>
- bans a user
'add [USER] <ROLE>
- gives a user a selected role
'remove [USER] <ROLE>
- removes a selected role from a user
'purge [NUMBER]
- purges a selected number of messages from current channel
